탐스러운 붉은빛을 가진
무농약 토마토

먹기 좋게 잘라 본연의 맛 그대로 즐기기도 하고, 으깨어 끓여 파스타 소스로 만들어 먹기도 하고, 갈아서 주스로 마시기도 하죠. 토마토의 선명한 붉은 색은 라이코펜이라는 영양소 때문인데, ‘라이코펜’은 베타카로틴, 루테인과 함께 과일과 채소에 색깔을 입히는 3대 영양소인데요. 토마토는 라이코펜 덩어리라고 해도 과언이 아니죠. 어떻게 먹든 껍질째 먹게 되는 채소이니 건강하게 무농약으로 드시는 것이 좋겠죠?

더욱 맛있게 즐기는 토마토

토마토는 100g당 14~17Kcal로 식단 조절에 도움을 주는 저칼로리 채소죠. 수분이 풍부해 포만감이 높기도 하고요. 토마토는 녹색이 남아있을 때 미리 따서 천천히 익히는 채소예요. 토마토를 살짝 익히면 영양이 더 잘 흡수된다고 하죠. 끓는 물을 부어 줘 살짝 데치거나, 고기를 구울 땐 구워서 같이 곁들여 보세요. 이제까지 몰랐던 토마토의 새로운 맛을 경험할 수 있으실 거예요.

좀 더 부드럽게 즐기고 싶다면 껍질을 벗겨 내고 먹는 방법도 있죠. 토마토를 뒤집어 작은 과도로 길게 십자 모양을 내어, 끓는 물에 잠깐 넣었다 빼 주세요. 얇은 껍질이 도르르 말리듯 일어날 거예요. 손으로 벗기면 아주 잘 벗겨진답니다. 부드러운 토마토의 식감을 즐기고 싶으시거나, 위장이 약한 어린아이들에게 주기에 좋은 방법이겠죠?
